# tweetie
Simple Social Medai System

To Start this application,
git clone to it

and type 
"python manage.py migrate"

to create superuser for admin panel login
"python manage.py createsuperuser"

to run the server
"python manage.py runserver"

requirements 
=============
django 2.2 or newer
pillow
crispy-form
rest-framework
